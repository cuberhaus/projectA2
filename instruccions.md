Executar en terminal:

MacOS:
```bash
brew install pip3
pip3 install networkx
pip3 install matplotlib
pip3 install numpy
pip3 install tqdm
python3 graph.py
```

Arch-based:
```bash
sudo pacman -S python-pip
pip3 install networkx
pip3 install matplotlib
pip3 install numpy
pip3 install tqdm
python3 graph.py
```
